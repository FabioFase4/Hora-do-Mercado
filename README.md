# API para Clientes e Produtos
# Objetivos:
- Simular uma API utilizando os conceitos de Orientação a Objetos e Conexão Front-Back
- Realizar Telas de Login, Cadastro, Visualização de Produtos, Carrinho e Vendas
- Não é um produto oficial

# Como usar:
- Use os serviços Apache e MySql do Xampp em conjunto com o MySql
- Utilize o Modelo de Banco de Dados disponibilizado em api/database/Banco.sql
- Abra o arquivo index.php no Navegador (Não como arquivo direto)
- http://127.0.0.1:8080/Mercado

# O que fazer?
- Ao abrir, irá aparecer opções para Acesso ou Abrir Githubs
- Execute os passos mostrados
